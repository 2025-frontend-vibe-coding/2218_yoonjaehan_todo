# 브랜드 컬러 가이드

## 선택된 컬러: 모던 퍼플

현재 적용된 컬러는 **모던 퍼플**입니다.

### 컬러 의미
- **AI 혁신**: 첨단 기술과 인공지능을 상징
- **창의성**: 혁신적이고 독창적인 서비스
- **프리미엄**: 고급스럽고 현대적인 느낌

### 컬러 코드
- **Primary**: `#8B5CF6` (Violet 500) - `oklch(0.6 0.18 300)` - 주요 액션 버튼, 링크
- **Secondary**: `#6366F1` (Indigo 500) - `oklch(0.55 0.15 250)` - 보조 요소
- **Accent**: `#A78BFA` (Violet 400) - `oklch(0.7 0.15 300)` - 강조 요소
- **Light Mode**: 밝고 생동감 있는 퍼플 톤
- **Dark Mode**: 부드럽고 세련된 퍼플 톤으로 눈의 피로 감소

## 대안 컬러 옵션

### 옵션 1: 프로페셔널 블루
**특징**: 신뢰, 집중, 전문성
- **Primary Blue**: `oklch(0.5 0.15 250)`
- **적합한 경우**: 비즈니스 환경, 전문가 타겟

### 옵션 3: 프레시 그린
**특징**: 성장, 완료, 긍정적 에너지
- **Primary Green**: `oklch(0.55 0.15 150)`
- **적합한 경우**: 동기부여와 성취감을 강조하고 싶을 때

## 컬러 변경 방법

1. `globals.purple.css` 또는 `globals.green.css` 파일을 열어주세요
2. 주석 처리된 컬러 값을 복사하세요
3. `globals.css`의 `:root`와 `.dark` 섹션에 해당 값들을 적용하세요

## 컬러 사용 가이드

### Primary Color 사용
- 주요 버튼 (추가, 저장, 확인)
- 중요한 링크
- 활성 상태 표시
- AI 기능 강조

### Accent Color 사용
- 호버 상태
- 보조 액션
- 배지 및 태그

### Muted Color 사용
- 배경
- 비활성 요소
- 구분선
